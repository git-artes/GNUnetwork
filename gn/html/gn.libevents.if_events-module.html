Module if_events

source code

Interface for libevents library with string based test frames.

This module is an interface to the events module.

To create an event object use function mkevent(). This function creates events of different types, according to the event modules imported by this module.

To add different types of events:

  • import library of specific event type.
  • add to function mkevent as necessary.

mkevent(nickname, **kwargs)

source code 

Returns an event of the given event nickname.

Parameters:
  • nickname - a valid event nickname, i.e. one that is a key in dictionary of valid nicknames.
  • kwargs - a dictionary of variables depending on the type of event. Field ev_dc is a dictionary of fields and values for the corresponding event type; field frmpkt is a binary packed frame.
Returns:
an Event object.

evtofrm(evobj, fr_dc_fldvals={}, fr_dc_frbd_fldvals={})

source code 

Make a Frame object from an Event object.

Dictionaries of field values dc_fldvals, dc_frbd_fldvals allow to adjust values in frame object fields and in frame body fields of frame object, respectively.

Parameters:
  • evobj - an Event object.
  • fr_dc_fldvals - dictionary of field values for frame object, default {}.
  • fr_dc_frbd_fldvals - dictionary of field values for frame body in frame object, default {}.
Returns:
a Frame object.
